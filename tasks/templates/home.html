Django App

This Django application was created on July 24, 2023, by Juan Carlos Salazar Mendoza. Its main purpose is to allow registered users to perform CRUD (Create, Read, Update, Delete) operations tasks. Users must first sign in to access the application, and they can later log out. Once registered and login, users have the ability to create new tasks, view existing tasks, edit them, save changes, and delete tasks. Additionally, they can mark tasks as important. The web project was developed using a combination of technologies, including HTML, CSS, Python, JavaScript, and Bootstrap. To host the application, used render.com. For data storage, the application uses PostgreSQL as its database, where tasks, user information, and session details are stored securely.
